Codes intepretion:

distribution.py illustrates exploring data analysis, which provides basic data distribution and allows for further preprocessing.

XGBALG.py illustrates how I use XGBoost and adjust the threshold to get different options of results. It includes the balance between accuracy and positive recall.

SHAP.py provides both shap and xgboost feature importance analysis. It guides how we weight the features.

Clustering.py outputs the K-means results and provide t-SNE Projection of Subscribers. For further visualization analysis of features (how we differ the clusters), please go to the Tableau: https://public.tableau.com/app/profile/yuhao.huang/viz/ProjectB_17648627756280/1_1?publish=yes
